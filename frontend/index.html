Como funciona

  1. Imprima este QR Code e cole no balcão da padaria
  2. O cliente escaneia com a câmera do celular
  3. O WhatsApp abre com a mensagem pré-preenchida
  4. O cliente envia — e é cadastrado automaticamente
  5. Na próxima fornada, ele já recebe o aviso!

Configure o número do WhatsApp Business no .env: PADARIA_NUMERO=5511999999999

Conectar WhatsApp
Webhook (cadastro automático)

Para que clientes sejam cadastrados automaticamente ao responder o QR Code, o webhook precisa estar configurado.

Clique em "Carregar QR"

Abra o WhatsApp no celular → Dispositivos conectados → Conectar dispositivo

Como instalar a Evolution API
1

Instale o Docker

A Evolution API roda via Docker. Instale em docker.com/get-started

2

Suba a Evolution API

docker run -d --name evolution-api -p 8080:8080 -e AUTHENTICATION_API_KEY=sua_chave atendai/evolution-api:latest

3

Configure o .env

Copie .env.example para .env e preencha EVOLUTION_API_URL, EVOLUTION_API_KEY e EVOLUTION_INSTANCE.

4

Escaneie o QR Code

Clique em "Carregar QR Code" acima e escaneie com o WhatsApp do número da padaria.

5

Em produção: use ngrok ou VPS

Para o webhook funcionar e receber cadastros automáticos, seu servidor precisa ser acessível publicamente. Use ngrok http 3000 para testes ou suba em uma VPS.
